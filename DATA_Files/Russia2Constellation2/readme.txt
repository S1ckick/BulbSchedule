Файлы с расчетными интервалами пролетов территории РФ спутниками группировки для местных времен с 09:00 до 18:00.
Во время этих интервалов времени сответствующие аппараты находятся над территорией России и могут снимать - соблюдаются оптимальные условия освещенности территории Солнцем.

Файл Russia-To-Satellite-SatPlanes_1_5 - интервалы для аппаратов типа Киноспутник (высокодетальная съемка)
Пять плоскостей с 10 аппаратами в каждой из них, всего 50 спутников
Аппараты Kinosat_110101 .. Kinosat_110510


Файл Russia-To-Satellite-SatPlanes_6_20 - интервалы для аппаратов типа Зоркий (съемка среднего разрешения)
Пятнадцать плоскостей с 10 аппаратами в каждой из них, всего 150 спутников
Аппараты Kinosat_110601 .. Kinosat_112010
